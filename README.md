##### Description:
###### This project focuses on analysing global mortality rates associated with unsafe water and sanitation. The goal was to uncover patterns and communicate insights through an interactive data visualisation dashboard.
##### Tools & Technologies:
###### Python / Excel / (Power BI if used – optional)
###### Data Cleaning & Visualisation
##### Key Tasks:
###### Cleaned and merged multiple CSV datasets to create a unified dataset
###### Handled missing values and ensured data consistency
###### Analysed trends in mortality rates across different regions
###### Designed a dashboard to visualise global health patterns
###### Presented findings and insights through a group presentation
##### Key Insights:
###### Higher mortality rates are concentrated in regions with limited access to clean water
###### Strong relationship between sanitation conditions and public health outcomes
###### Data highlights the need for improved infrastructure and policy interventions
##### Outcome:
###### Created a data-driven dashboard to support awareness and decision-making in public health and sanitation.
